## Elevator pitch

**ExtractorX** extracts every embedded image from Word, PowerPoint, and PDF documents — organized by format, deduplicated, and reported — in a polished desktop product built for agencies and enterprise document workflows.

---

## Highlights for portfolio

- Multi-format extraction (DOCX, DOC, PPTX, PPT, PDF)
- Batch processing with real-time progress
- SHA-256 duplicate detection
- Branded TXT, PDF, Excel, and metadata reports
- Multilingual UI (EN / FR / AR) with dark/light themes
- 100% local processing — no cloud upload
